### О проекте

**Фреймворк**
- FastAPI

**База данных**
- Postgrsql

**ОРМ**
- Ormar + Alembic

### Для запуска

- клонировать проект
- перейти в дерикторию с проектом

**В докере**
- в терминале выполнить команду `docker-compose up`

**На локальной машине**
- перейти в дериктоию app
- настройки для бд в файле database/config.py
- в терминале выполнить команду `uvicorn main:app --reload`
